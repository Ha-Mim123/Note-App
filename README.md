# 📝 Notes App (React)

A simple and responsive Notes App built with React. Users can add, edit, and delete notes with data saved in localStorage.

## 🚀 Features
- Add, edit, delete notes  
- Data persistence (localStorage)  
- useState for state management  
- useRef for input focus  
- Responsive design  
- Clean UI  

## 🛠️ Tech Stack
React, JavaScript, HTML, CSS
